# Prompt 网站分类
## 文本类

主要用于文本生成模型（如 ChatGPT, Claude, 文心一言等），提供角色预设、任务指令和对话模板。

| 网站/资源名称              | 说明描述                                | 访问链接                                                         |
| -------------------- | ----------------------------------- | ------------------------------------------------------------ |
| **ChatGPT Shortcut** | 结构化的 ChatGPT 快捷指令表，支持中英文搜索和分类。      | [点击访问](https://newzone.top/chatgpt/)                         |
| **FlowGPT**          | 全球最大的 Prompt 分享社区，包含各种生产力工具和角色扮演模板。 | [点击访问](https://flowgpt.com/)                                 |
| **PromptHub**        | 精选的提示词库，提供测试和版本管理功能（WaytoAGI 推荐）。   | [点击访问](https://www.prompthub.us/)                            |
| **Snack Prompt**     | 每日更新的 Reddit 风格提示词社区，突出热门和实用的指令。    | [点击访问](https://snackprompt.com/)                             |
| **ChatGPT 中文调教指南**   | GitHub 上的开源项目，整理了大量适合中文语境的 Prompt。  | [点击访问](https://github.com/PlexPt/awesome-chatgpt-prompts-zh) |
| **AIPRM (插件)**       | 集成在 ChatGPT 网页端的扩展程序，提供成千上万的现成指令。   | [点击访问](https://www.aiprm.com/)                               |

## 图像类

主要用于 AI 绘画模型（如 Midjourney, Stable Diffusion, DALL-E 等），提供关键词、风格参考和参数提示。

| 网站/资源名称 | 说明描述 | 访问链接 |
| --- | --- | --- |
| **Lexica** | 专门针对 Stable Diffusion 的搜索引擎，可查看图片及其完整 Prompt。 | [点击访问](https://lexica.art/) |
| **PromptHero** | 专业的 AI 摄影和艺术提示词平台，涵盖 MJ、SD 和 DALL-E。 | [点击访问](https://prompthero.com/) |
| **Arthub.ai** | 展示 AI 生成的艺术作品，并分享其背后的提示词与模型参数。 | [点击访问](https://arthub.ai/) |
| **Krea.ai** | 实时生成和提示词辅助工具，特别适合寻找视觉风格。 | [点击访问](https://www.krea.ai/) |
| **MJ Prompt Tool** | Midjourney 辅助工具，通过选择各种风格标签快速组合咒语。 | [点击访问](https://prompt.noonshot.com/) |
| **Civitai (C站)** | 模型分享社区，每个模型预览图下方通常带有详细的生成参数。 | [点击访问](https://civitai.com/) |

## 方法论与 Github 项目

提供提示工程（Prompt Engineering）的理论学习、高级技巧以及开发者相关的开源工具。

| 项目/资源名称 | 说明描述 | 访问链接 |
| --- | --- | --- |
| **Prompt Engineering Guide** | 全球最著名的提示工程指南，提供系统性的学术和实践教学。 | [点击访问](https://www.promptingguide.ai/zh) |
| **Learn Prompting** | 免费、开源的提示工程课程，适合从入门到进阶。 | [点击访问](https://learnprompting.org/zh-Hans/) |
| **Awesome ChatGPT Prompts** | GitHub 明星项目，汇集了所有主要角色的英文 Prompt 指令。 | [点击访问](https://github.com/f/awesome-chatgpt-prompts) |
| **LangGPT** | 结构化提示词方法论，通过类似编程的格式提高 Prompt 成功率。 | [点击访问](https://github.com/yzfly/LangGPT) |
| **OpenAI Cookbook** | OpenAI 官方提供的代码案例，包含大量 Prompt 优化示例。 | [点击访问](https://github.com/openai/openai-cookbook) |
