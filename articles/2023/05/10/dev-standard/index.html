提交注释一览

类型 描述
feat 新特性、新功能
fix 修改bug
ci 持续集成修改
docs 文档修改
style 代码风格或格式修改
refactor 重构代码
perf 性能优化等
test 测试用例
build 编译构建工具改动
revert 回滚代码版本
chore 其他修改：比如改变构建过程或辅助工具的变动，增加依赖库等
